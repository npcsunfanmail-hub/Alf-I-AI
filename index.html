TV Settings

Not connected
Scans for TV via Bluetooth, then auto-finds its IP
App-store remote apps use WiFi (not Bluetooth) to send commands. This scans for your TV via Bluetooth to identify it, then finds its IP and uses WiFi for control. For Samsung: Settings → General → External Device Manager → Device Connect Manager → "Always Allow". For LG: Settings → Connection → Mobile TV On → On.
